# 13 - Moment Blurb Guide

## Purpose

Generate short public blurbs for detected moments. A moment blurb explains why one hand mattered without pretending it was a full article.

## Shape

- Name the hand.
- Name who won the pot.
- State the pot or result if supplied.
- Explain the consequence in one or two sentences.

## Do

- Keep it sharp.
- Use hand number, pot size, winner, board, and winning hand only when supplied.
- Make the moment readable for a player.

## Do Not

- Invent action, reads, bluffs, emotion, or rivalry.
- Overstate small pots.
- Reuse the same sentence pattern for every moment.
