# Player Archetypes

## Purpose

Player archetypes translate poker tendencies into respectful public profile language.

They are not insults, diagnoses, or scouting leaks. They should help a player feel recognized while keeping advanced HUD/scouting analysis for future internal or paid-tier surfaces.

## Public Language Rules

- Use style language, not judgment language.
- Avoid calling players bad, scared, reckless, lazy, clueless, or weak.
- Avoid overclaiming from small samples.
- Prefer "listed as", "profiled as", "showing early signs of", or "currently tagged as" when data is thin.
- Do not expose sensitive advanced stats by default.

## Translation Table

| Raw poker label | Public-facing archetype | Optional subtitle |
| --- | --- | --- |
| Loose Aggressive | Pressure Builder | High-engagement player who keeps pots active |
| Loose Aggressive | Initiative Driver | Creates action and tests table resistance |
| Loose Passive | Showdown Seeker | Willing to see more boards and let hands resolve |
| Loose Passive | Wide-Range Regular | Plays a broad table presence without needing every pot to be forced |
| Tight Aggressive | Selective Pressure | Chooses spots carefully, then applies force |
| Tight Aggressive | Precision Opener | Lower-volume profile with clear initiative when engaged |
| Tight Passive | Selective Anchor | Patient presence with a lower-variance public profile |
| Tight Passive | Quiet Defender | Lets the table move before committing public pressure |
| Calling Station | Showdown Loyalist | Difficult to push out once the hand has a story |
| Maniac | Volatility Engine | High-action style that can reshape a session quickly |
| Nit/Rock | Patient Anchor | Selective, durable, and hard to pull into noise |
| Shark | Table Closer | Converts pressure into results when the record supports it |
| Fish | New/Developing Player | Never use "fish" publicly |

## Current S0 Examples

Para-Poker:

- Raw label: Loose Aggressive / Pressure Player
- Public label: Pressure Builder
- Subtitle option: "High-engagement profile with early Season 0 pressure markers."

panicmixie:

- Raw label: Loose Passive / Showdown Closer
- Public label: Showdown Seeker
- Subtitle option: "Wide-range table presence with late-hand staying power."

## Badge/Chip Language

Good:

- Preseason Regular
- Pressure Builder
- Showdown Seeker
- Season 0 Starter
- First Approved Win
- Biggest Listed Pot
- Final Hand Feature
- Session Runner-Up

Avoid:

- Donkey
- Fish
- Choker
- Maniac, unless player-approved as a playful title
- Station, unless translated
- Whale
- Degenerate
- ATM

## Future Paid/Internal Layer

Advanced HUD/scouting can preserve raw poker concepts internally:

- VPIP
- PFR
- aggression frequency
- showdown stats
- position behavior
- hand class tendencies

Public v2 should not turn those into shaming labels.
