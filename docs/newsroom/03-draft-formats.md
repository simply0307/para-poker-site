# Draft Formats

## Session Recap

Purpose: public article for a completed session.

Fields:

- headline
- subheadline
- recap_body
- key_moments
- player_blurbs
- confidence_notes
- missing_data_warnings

Confidence notes and missing data warnings are admin-only.

## Player Profile

Purpose: public player-facing identity page.

Use only approved league data and reviewed copy.

## Moment Blurb

Purpose: short public explanation of why one hand mattered.

Use hand number, pot, winner, and consequence only when supplied.

## Standings Summary

Purpose: explain the approved standings in plain league terms.

Do not infer movement without before/after standings.

## Article

Purpose: broader league coverage from structured data and editorial request.

If the request is under-specified, return missing data warnings rather than inventing.
